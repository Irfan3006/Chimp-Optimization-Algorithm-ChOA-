## Penjelasan Kode
Program ini (`main.py`) adalah sebuah Sistem Pendukung Keputusan (SPK) untuk memilih paket internet terbaik dari berbagai provider. Program ini menggunakan **Chimp Optimization Algorithm (ChOA)** untuk mengoptimalkan bobot dari berbagai kriteria pemilihan (Harga, Kuota, Masa Aktif, Jaringan, dan Bonus). Data paket internet dinormalisasi dan dihitung skor akhirnya berdasarkan bobot terbaik yang ditemukan oleh algoritma ChOA, lalu ditampilkan peringkat paket internet dari yang paling direkomendasikan.
